사이드바 기준 사용자별 시뮬// +예측, 알림, 게시판
//농부
1. 메인화면 - Dashboard
예상 수익금, 요청/예상 생산량, 판매량 확인 가능


2. 예측 재고 등록 화면 - Inventory Register
자동으로 재고를 등록해주니까 필요 없을듯?
- 재고ID, 작물 종류, 상태, 예측 산출량, 예측 수확일, 단가


3. 재고 관리 화면 - Inventory
- 재고ID, 작물 종류, 상태, 예측 산출량, 최대 산출량, 예측 수확일, 수정 버튼
	수정 버튼 누르면 재고 수정하는 페이지 이동
		하나의 재고당 재고 수정 화면 (재고 수정)
		예측 산출량, 예측 수확일 수정 가능
		재고 삭제 버튼


4. 주문 관리 화면 - Farm Order Detail
- 주문ID, 주문자, 가격, 주문 수량, 배송 상태, 주문 날짜
	하나의 주문 누르면 구매자가 작성한 상세 화면으로 이동
		

5. 알림 화면
칸으로 나뉘어서 주문, 생육, 수확, 재고 알림 칸 으로 구분
- 주문 수락 알림 화면
	구매자의 주문 생성 요청을 수락/거부 할 수 있어야 함
- 생육 이상 알림 화면
- 수확 예정 알림 화면
- 재고 부족 알림 화면
